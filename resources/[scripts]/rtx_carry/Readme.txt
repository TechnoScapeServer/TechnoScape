Install instructions:
1. Rename rtx_carry.pack to rtx_carry
2. Put rtx_carry folder to your resources
3. Configure your config.lua to your preferences
4. If you want use new carry style, remove old folder html and rename folder html_new to html
5. Put rtx_carry to the server.cfg
